# The 6 Most Important Assertion Macros

| Macro                          | When to Use            |
| ------------------------------ | ---------------------- |
| `assert!(cond)`                | Check a condition      |
| `assert_eq!(a, b)`             | Strict equality        |
| `assert_ne!(a, b)`             | Ensure things differ   |
| `assert_matches!()`            | Pattern matching tests |
| `matches!()`                   | Quick pattern testing  |
| `panic!()` + `#[should_panic]` | Test crashing cases    |
